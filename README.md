# Turkmen Gas History

An interactive chronological atlas of Turkmenistan's natural gas industry from 1950 to 1976, plotted on a modern satellite base.

→ **Live map: https://cnwizda.github.io/Turkmen-gas-history/**

The map covers 31 features across five time periods, from the first gas gusher at the Gyzylgum field (1952) through the Bukhara–Ural and Central Asia–Center I trunk pipelines (1966–67). Each feature includes article-sourced descriptions, key facts, and full coordinate data.

## Sources

- O. Kheshdekov, "Early history of natural gas in Turkmenistan," *News Central Asia*, 22 May 2024. https://www.newscentralasia.net/2024/05/22/early-history-of-natural-gas-in-turkmenistan/
- USGS Bulletin 2201-H (Ulmishek 2004) — Amu-Darya basin geology
- Wikipedia, Wikidata, OpenStreetMap — coordinate verification
- Esri World Imagery — basemap tiles

## Methodology

Each feature has been geolocated against authoritative sources and tagged with a confidence level (HIGH / MEDIUM / LOW). Low-confidence features (notably some 1962–65 Aral expedition drill sites whose exact well coordinates are not in modern open sources) are visually distinguished with dashed marker outlines and a "(~)" suffix on the label.
